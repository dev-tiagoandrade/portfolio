Jornal Virtual | O Federado

“O Federado” é um projeto de jornal virtual colaborativo, criado com a finalidade de proporcionar uma plataforma de expressão para a comunidade acadêmica do IFPA Campus Belém. Foi desenvolvido como parte do Trabalho de Conclusão do curso de Desenvolvimento de Sistemas no IFPA.

Nesse projeto, minha responsabilidade foi a criação de um protótipo de interface, seguindo as práticas de Experiência do Usuário e Usabilidade de Donald Norman, além do Desenvolvimento Front End do Site.
